# Proposal PDF Generator

Веб-приложение для генерации коммерческих предложений в PDF **на клиенте** (без сервера) с live preview.

## Стек

- React + TypeScript + Vite
- Tailwind CSS
- React Router v6
- @react-pdf/renderer
- React Context + useReducer + localStorage

## Запуск

Установите зависимости и запустите dev-сервер:

```bash
npm install
npm run dev
```
